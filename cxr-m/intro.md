# CXR-M SDK Introduction

_Source: https://custom.rokid.com/prod/rokid_web/57e35cd3ae294d16b1b8fc8dcbb1b7c7/ (Chinese, fetched 2026-05-29)._

The Rokid CXR-M SDK is a developer toolkit provided by Rokid for building mobile applications that work with Rokid Glasses.

See [release-notes.md](release-notes.md) for the full version history. The current **publicly documented** release is **v1.1.0** (2026-04-01); Maven additionally publishes **v1.2.1** (uploaded 2026-04-17) but Rokid has not released an official changelog for the 1.1.x / 1.2.x line as of 2026-05-29.

> **Note:** The CXR-M SDK is not publicly distributed on the Rokid developer site. To request the SDK, documentation, or technical support, contact business partnerships at `Glasses.BD@rokid.com`.

## Capabilities

### Rokid Glasses AI Process Interaction

Developers can quickly integrate the AI interaction flow defined by the YodaOS-Sprite operating system using the CXR-M SDK to implement custom AI workflows and scenes.

### Rokid Glasses Hardware Information

Developers can quickly obtain hardware information about Rokid Glasses using the CXR-M SDK.

### Rokid Glasses Assist Service

Developers can use the CXR-M SDK to access services provided by the Rokid Assist Service, including file transfer, audio recording, and photo capture.

> **Note:** The Rokid Assist Service is a set of system services built on top of YodaOS-Sprite. These services are only accessible through the CXR-M SDK when they are enabled on the device. If Assist Service is disabled, the corresponding SDK features will be unavailable.
